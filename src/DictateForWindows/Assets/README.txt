DICTATE FOR WINDOWS - ASSETS

This directory should contain the following assets:

Required for WinUI 3:
- app-icon.ico          - Application icon (256x256, multi-size ICO)
- app-icon.png          - Application icon PNG (1024x1024)
- StoreLogo.png         - Store logo (50x50)
- Square44x44Logo.png   - Taskbar icon (44x44)
- Square150x150Logo.png - Start menu tile (150x150)
- Wide310x150Logo.png   - Wide tile (310x150)
- LargeTile.png         - Large tile (310x310)
- SplashScreen.png      - Splash screen (620x300)

Optional:
- recording-indicator.png - Recording animation frames
- microphone.png          - Microphone icon
- settings.png            - Settings icon

Icon Design Guidelines:
- Primary color: #6366F1 (indigo)
- Recording color: #EF4444 (red)
- Use simple microphone + waveform motif
- Ensure icons are legible at small sizes

To generate assets, use tools like:
- Visual Studio Image Asset Generator
- Inkscape/Illustrator for SVG source
- ImageMagick for batch conversion
